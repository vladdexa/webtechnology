Descrierea proiectului

OTO(online toys) este o aplicatie web ce consta in gestionarea colectiilor de jucarii de diferite varste si diferite categorii. Sistemul ofera suport pentru activitati de cumparare a jucariilor si administrarea utilizatorilor. De asemenea, sistemul ofera posibilitatea de a genera statistici exportate in format PDF si CSV. Se poate realiza si un clasament al celor mai populare produse, disponibil si ca flux de date RSS.

Descrierea paginilor

Pagina autentificare

Register ofera posibilitatea vizitatorilor de a se inregistra pe site completand un formular scurt cu datele personale pentru ca mai apoi sa fie posibila conectarea acestuia prin sistemul de login.Acestea sunt necesare pentru ca utilizatorii sa poata beneficia de optiunile noastre de vizualizare produse, adaugare in cos, plasare comanda.
Este implementat si sistemul de ForgotPassword care ofera posibilitatea utilizatorului sa-si recupereze parola cu ajutorul emailului.

Pagina Acasa

Dupa logare, utilizatorii sunt redirectionati catre pagina Home ce contine o parte din produsele disponibile pe site dar si un navbar simplificat usor de utilizat si foarte folositor. Acest navbar ofera acces la statistici PDF si CSV legate de produsele noastre, ofera de asemenea posibilitatea de a fi redirectionat pe pagina de categorii, detalii utilizator si vizualizarea cosului de cumparaturi.

Pagina Categorii

In aceasta pagina sunt prezentate produsele provenite doar dintr-o anumita categorie selectata de utilizator.

Pagina User

In pagina utilizatorului sunt prezentate detaliile acestuia, mai exact username-ul si email-ul cu care acesta s-a conectat pe site-ul nostru.
Tot aici este prezenta si posibilitatea utilizatorului de a-si schimba parola prin completarea formului "change your password".

Pagina de comanda

Order Page este pagina unde utilizatorul poate finaliza procesul de cumparare a jucariilor prezente in cosul de cumparaturi, prin completarea unui form cu detaliile de plata si adresa.
